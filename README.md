## ğŸ› How To Report A Bug

A bug is when something:
- Does not work
- Crashes
- Responds incorrectly
- Behaves differently than expected

### Step-by-step

1. Click the **Issues** tab.
2. Click the green **New Issue** button.
3. Select **Bug report** (or create a blank issue if templates are not enabled).
4. Fill in:

   - **What happened?**
   - **What did you expect to happen?**
   - **Steps to reproduce it**
   - Screenshots (if possible)
   - The command you used (if relevant)

### Good example
Command: /start
Expected: Game should begin
Actual: Bot does not respond
Steps: Used in #general, no permissions changed


The more details you give, the faster we can fix it.
